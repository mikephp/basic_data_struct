Hinzufügen einer neuen Applikation
Der einfachste Weg um eine neue Applikation hinzuzufügen ist sie einfach mit der Maus vom Desktop oder Windows Explorer in das Freecap Programmfenster zu ziehen.
Aber es geht auch anders. Sie können ein eigenes Profil für die Applikation erstellen.
Drücken Sie den Knopf "New profile" im Werkzeugpanel, oder selektieren Sie aus dem Menü "File->New application". Sie sollten nun ein Fenster wie dieses sehen:
Hier können Sie den Pfad zur Applikation manuell eingeben, oder mit "Browse" nach ihrer Applikation suchen.
Nachdem Sie die Applikation gewählt haben fügt FreeCap die restlichen Informationen automatisch ein.
