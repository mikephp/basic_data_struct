FreeCap
FreeCap wurde als Open Source Alternative zum "freeware" SocksCap erstellt. Mit FreeCap können Sie Applikationen die selbst keinen Proxy unterstützen mit einem Proxy Server verwenden. Die Hauptfunktionen von FreeCap sind:
  • Umwandlung eines SOCKS Proxy in einen HTTP(S) Proxy
  • Aneinanderreihen mehrerer Proxies sogar mit verschiedenen Protokollen (z.B. SOCKSv4 -> HTTP -> SOCKSv4 -> SOCKSv5)
  • Genaue Umsetzung der RFC 1928, 1929, 2817 (SOCKS v4 und v5, Authorisierung für SOCKSv5 und HTTP CONNECT)
  • DNS-Auflösung über jeden Proxy
  • Schritt-für-Schritt DNS-Auflösung über die Proxy-Kette
Und noch viele andere Funktionen machen FreeCap zu einem wertvollen Werkzeug.
