Frequently Asked Questions (Oft Gestellte Fragen)
Q: Welche Arten von Netzwerkverkehr kann FreeCap über den Proxy schicken ?
A: FreeCap tunnelt TCP und UDP Verbindungen, wobei UDP nur über maximal einen SOCKSv5 Proxy ohne NAT funktioniert. Andere Protokolle (wie ICMP, IGMP etc) können technisch, oder laut RFC Standards nicht über einen Proxy geschickt werden.
Q: Warum funktioniert UDP über Proxy nicht über NAT ? (Oder: Warum kann man UDP Proxies nicht hintereinanderreihen?)
A: Weil FreeCap für UDP den Ursprungs-Port wissen muss. Wenn dazwischen aber NAT gemacht wird geht diese Information verloren, da NAT die Datenpakete verändert. Das ist auch der Grund warum UDP nicht über mehrere Proxies laufen kann.

Q: Ich habe einen funktionierenden HTTP Proxy, aber kann trotzdem auf keine Internetseite über den Proxy zugreifen. Der Proxy funktioniert im Internet Explorer aber nicht in FreeCap. Warum ?
A: Es gibt zwei Arten von HTTP Proxies, den sogenannten "caching" Proxy der keine durchgehenden Verbindungen erlaubt sondern nur oft aufgerufene Internetseiten zwischenspeichert, und HTTP Proxies die direkte Verbindungen zulassen. Freecap benötigt einen HTTP Proxy der die "CONNECT method" unterstützt, also eine direkte Verbindung durch den Proxy erlaubt..
Q: FreeCap funktioniert nicht ! Ich benutze das Programm XXX aber die Verbindung wird verweigert ! ("Connection refused" oder "Page cannot be displayed", etc)
A: Mehrere mögliche Gründe:
  • Ist der Proxy noch aktiv ?
  • Versuchen Sie über den Internet Proxy auf einen Computer innerhalb der lokalen Netzwerkes zuzugreifen ?
  • Benutzt ihr Internet Provider einen transparenten Proxy? (Häufiger Grund !)
  • Haben sie auf die Authentifizierung (User/Pass) am Proxy vergessen ?
  • Wenn sie immer noch nicht wissen wo das Problem liegt aktivieren Sie das Log in FreeCap um mehr Informationen zu bekommen.
Q: FTP funktioniert nicht ! Ich kann eine Verbindung zum Server herstellen, aber ich sehe keine Ordner/Dateien !
A: Sie müssen den "passive mode" in ihrem FTP Programm aktivieren. Dies sollte in der Hilfe des FTP Programmes erklärt sein.
Q: FreeCap stürzt ab !
A: Kontaktieren mich bitte ! Ich benötige die folgenden Informationen :
  • OS (Windows Version) und Service Pack
  • Welches Programm stürzt ?
  • Bildschirmfoto von der Fehlermeldung wenn vorhanden.
  • Gepackte (Winzip/Winrar) FreeCap Log Datei
  • Genaue Informationen über die Aktivität die zum Absturz führte.
