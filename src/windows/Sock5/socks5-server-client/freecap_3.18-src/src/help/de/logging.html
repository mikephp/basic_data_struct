Konfiguration — Log
Log Dateien sind oft hilfreich wenn Probleme auftreten, oder auch einfach nur um mitzuschreiben was für Daten von einem Programm übertragen werden. Bitte denken Sie daran diese Log Dateien mitzuschicken wenn Sie mich wegen eines Problems in FreeCap kontaktieren.
Hook injectionAlles über injection/hooking
ConnectionAktivitäten des Programms über WinSock
SOCKSAlle SOCKS Aktivitäten (SOCKSv4, SOCKSv5 und HTTP)
Warning messagesohne Worte :)
Log trafficÜbertragene Daten in eigene Dateien mitschreiben. Dateiname ist gleich wie Log Name aber mit Suffix:
Für gesendete Pakete: ".<socket handle>.asend.<sequence number>"
Für erhaltene Pakete: ".<socket handle>.brecv.<sequence number>"
