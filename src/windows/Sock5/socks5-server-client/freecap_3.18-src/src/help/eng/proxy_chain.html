Config — Proxy chains
Proxy chains are power tool if you want to make work harder for "men in black" :)
HostHostname/IP of proxy
ProtoUsed protocol. Can be SOCKS4, SOCKS5 or HTTP
AnonAnonymous (Y) or not (N).
PingPing to proxy. To check ping -- select "Ping proxy" from context menu
StatusStatus of proxy. "OK" means that proxy responds
FreeCap doesn't contain full proxy checker, that means that "proto" and "anon" columns depends on records in config.

Fields are same as in Default Proxy tab.
