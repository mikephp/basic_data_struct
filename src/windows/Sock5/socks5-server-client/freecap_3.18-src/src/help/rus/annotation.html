FreeCap
Программа FreeCap создавалась как альтернатива "бесплатному" SocksCap-у и предназначена для соксификации программ не умеющих работать с прокси серверами самостоятельно. Из основных достоинств программы следует выделить:
  • Возможность работы как с SOCKS прокси, так и с HTTP(S) прокси
  • Возможность использовать цепочки разнородных серверов. (к примеру SOCKSv4 -> HTTP -> SOCKSv4 -> SOCKSv5)
  • Полная поддержка стандартов RFC 1928, 1929, 2817 (SOCKS v4 и v5, авторизация для SOCKSv5 и HTTP CONNECT)
  • Разрешение имен DNS через любую прокси
  • Последовательное разрешение имен DNS через цепочку прокси серверов
Эти и многие другие особенности делают FreeCap уникальной программой в своем роде.
