.Topic IDH_POPUP_PROGRAM_STARTUP
Запуск FreeCap вместе с Windows

.Topic IDH_POPUP_PROGRAM_RUNMINIZE
При запуске FreeCap автоматически сворачивается в трей.

.Topic IDH_POPUP_PROGRAM_MINIMIZE
Сворачивать FreeCap в системный трей (рядом с часами).

Добавлять строчку "via FreeCap" к заголовкам соксифицируемых приложений.

.Topic IDH_POPUP_PROGRAM_SHOWMESS
Показывать сообщения об ошибках в диалоговом окошке. Если галочка снята, сообщения будут писаться в лог-файл.

.Topic IDH_POPUP_PROGRAM_DNS
